CLASIFICACIÓN DEL SOFTWARE

  1. Sofware de sistema: Es aquel que nos permite interactuar directamente con el hardware de nuestro ordenador. Por ejemplo:
    • Sistemas operativos: Windows, Linux, Android, etc.
    • Controladores de dispositivo (Drivers)
  2. Software de programación: Conjunto de aplicaciones mediante las cuales se pueden desarrollar (crear) programas informáticos.
  3. Software de aplicación: Se trata de aquellos programas que permiten al usuario realizar tareas específicas en el sistema, existen diferentes categorías:
    • Ofimáticas: Microsoft Office, Libreoffice, etc.
    • Multimedia: VLC, Reproductor de Windows, Winamp, etc.
